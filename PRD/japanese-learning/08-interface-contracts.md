# 交互契约

本文档使用“命令、查询、事件”的业务语义描述交互契约，不限定通信协议、交互方式或具体工具。

## 外部输入

| 输入 | 来源 | 语义 | 必要字段 |
| --- | --- | --- | --- |
| 资源查询条件 | 用户 | 筛选日语学习资源 | 分页信息、可选筛选条件 |
| 资源表单 | 用户 | 新增或编辑日语资源 | 资源字段 |
| 资源标识集合 | 用户 | 批量删除、生成音频、翻译或合成课程音频 | 一个或多个资源标识 |
| 日语句子输入 | 用户 | 学习游戏录入 | 日语文本 |
| 当前日期 | 系统环境 | 判断今日统计是否过期 | 日期 |
| 翻译结果 | 外部语言能力 | 日语转中文 | 原文、译文、原文音频、译文音频 |
| 朗读生成结果 | 外部语言能力 | 文本转音频 | 音频资源 |

## 外部输出

| 输出 | 接收方 | 语义 |
| --- | --- | --- |
| 资源列表 | 用户 | 展示查询结果 |
| 资源详情 | 用户 | 展示完整资源信息 |
| 导出文件 | 用户 | 离线使用或备份资源 |
| 基础卡片矩阵 | 用户 | 展示基础日语内容 |
| 单条播放音频 | 用户 | 听日语原文 |
| 合成课程音频 | 用户 | 连续听力训练 |
| 学习统计 | 用户 | 展示今日和累计成果 |
| 成功或失败提示 | 用户 | 反馈操作结果 |

## 命令

### CreateLearningResource

- 目的：创建日语学习资源。
- 输入：日语文本、中文含义、音频列表、来源。
- 输出：成功或失败。
- 幂等性：当前未确认。同一内容重复提交可能创建多条资源。
- 一致性要求：成功后资源可被查询到。

### UpdateLearningResource

- 目的：更新一条日语学习资源。
- 输入：资源标识和待更新字段。
- 输出：成功或失败。
- 幂等性：相同更新重复执行后结果应一致。
- 一致性要求：成功后详情和列表展示新值。

### DeleteLearningResources

- 目的：删除一条或多条资源。
- 输入：资源标识集合。
- 输出：成功或失败。
- 幂等性：重复删除已删除资源的结果语义待确认。
- 一致性要求：成功后资源不应出现在查询结果中。

### GenerateReadAudio

- 目的：为资源生成日语和中文朗读音频。
- 输入：资源标识集合。
- 输出：成功处理结果。
- 幂等性：当前表现为追加音频，重复执行可能产生多个音频。
- 一致性要求：单条成功后，对应音频列表新增音频。

### TranslateChineseMeaning

- 目的：根据日语文本补充中文含义。
- 输入：资源标识集合。
- 输出：成功或失败。
- 幂等性：相同日语文本重复翻译通常应得到相同或近似结果，但外部语言能力可能变化。
- 一致性要求：成功后中文含义可被查看。

### GenerateCourseAudio

- 目的：把选中资源合成为连续听力课程音频。
- 输入：资源标识集合。
- 输出：合成音频资源。
- 幂等性：相同资源重复合成可生成新的合成音频；是否复用旧结果待确认。
- 一致性要求：返回前应保证合成音频可访问。

### RefreshDailyLearningStats

- 目的：刷新今日学习统计。
- 输入：当前日期。
- 输出：最新学习统计。
- 幂等性：同一天重复刷新结果不变；跨日首次刷新会清零今日统计。
- 一致性要求：刷新后的今日日期应为当前日期。

### SubmitLearningResult

- 目的：保存一次游戏录入结果并更新统计。
- 输入：日语文本、中文含义、日语音频、中文音频、来源、本次耗时。
- 输出：成功或失败。
- 幂等性：当前未确认；重复提交可能重复创建资源并重复累计统计。
- 一致性要求：成功时资源保存和统计更新应同时成立；失败时不应出现资源与统计不一致。

### SwitchInputMode

- 目的：请求切换输入模式。
- 输入：目标输入模式。
- 输出：成功或失败。
- 幂等性：重复切换到同一模式结果应一致。
- 一致性要求：失败不影响学习提交主流程。

## 查询

### QueryLearningResources

- 输入：分页、资源标识、创建时间范围、来源。
- 输出：资源列表和总数。
- 一致性要求：应反映最近成功保存或删除的资源。

### GetLearningResourceDetail

- 输入：资源标识。
- 输出：资源详情。
- 一致性要求：不存在时返回空或错误语义，具体表现待确认。

### GetBasicLearningCards

- 输入：基础来源条件。
- 输出：卡片矩阵。
- 一致性要求：只包含基础来源资源和必要空占位。

### GetLearningStats

- 输入：无或统计范围。
- 输出：统计起点、今日数量、今日时长、累计数量、累计时长。
- 一致性要求：展示前应先应用跨日刷新规则。

### CountLearningResources

- 输入：查询条件。
- 输出：资源数量。
- 一致性要求：数量与相同条件下的查询结果一致。

### GetRelatedLearningLinks

- 输入：无。
- 输出：相关学习资料入口。
- 一致性要求：资料入口应可被用户打开；失效检测机制待确认。

## 事件

| 事件 | 触发条件 | 载荷 | 订阅者 |
| --- | --- | --- | --- |
| LearningResourceCreated | 资源创建成功 | 资源标识 | 列表、统计或审计能力 |
| LearningResourceUpdated | 资源更新成功 | 资源标识、变更字段 | 列表、详情 |
| LearningResourcesDeleted | 资源删除成功 | 资源标识集合 | 列表 |
| ReadAudioGenerated | 朗读音频生成成功 | 资源标识、音频摘要 | 资源详情、列表 |
| ChineseMeaningTranslated | 中文释义补充成功 | 资源标识、中文含义 | 资源详情、列表 |
| CourseAudioGenerated | 课程音频生成成功 | 合成音频摘要 | 用户下载或播放入口 |
| DailyStatsReset | 今日统计跨日重置 | 旧日期、新日期 | 统计展示 |
| LearningSubmissionCompleted | 游戏提交成功 | 资源标识、耗时、统计增量 | 统计展示、资源列表 |

## 错误语义

当前未发现稳定错误码体系，以下为业务语义，用于不同实现对齐：

| 错误语义 | 触发条件 | 建议用户反馈 |
| --- | --- | --- |
| VALIDATION_FAILED | 输入为空、格式不合法或超过限制 | 请检查输入内容 |
| RESOURCE_NOT_FOUND | 资源标识不存在 | 资源不存在或已删除 |
| STATS_NOT_FOUND | 学习统计记录不存在 | 未找到游戏统计信息 |
| TRANSLATION_UNAVAILABLE | 翻译能力不可用或失败 | 翻译失败，请稍后重试 |
| AUDIO_GENERATION_UNAVAILABLE | 朗读生成失败 | 音频生成失败，请稍后重试 |
| AUDIO_MISSING | 播放或合成所需音频缺失 | 当前资源缺少音频 |
| COURSE_AUDIO_EMPTY | 没有资源可参与课程音频合成 | 所选资源缺少必要音频 |
| CONFIRMATION_MISMATCH | 批量确认输入不匹配 | 输入框内容不匹配,请重新输入 |
| SUBMISSION_FAILED | 学习结果保存或统计更新失败 | 学习提交失败，请重试 |

## 幂等性要求汇总

- 查询类交互应天然幂等。
- 更新类交互重复执行同样字段值后，应保持相同资源结果。
- 音频生成和学习提交当前不具备确认的幂等语义，重复执行可能追加音频或重复累计。
- 跨日刷新在同一天内应幂等。

## 一致性要求汇总

- 游戏提交成功时，资源保存和统计更新应保持一致。
- 删除资源后，列表、详情和卡片视图不应继续展示已删除资源。
- 生成音频成功后，单条播放和课程合成都应能使用新增音频。
- 课程音频合成只应纳入满足音频条件的资源。
