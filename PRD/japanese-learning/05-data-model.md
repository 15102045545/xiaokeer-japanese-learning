# 数据模型

本文档描述业务数据模型，不限定任何存储、接口或技术实现方式。

## 实体一：日语学习资源

用于沉淀可查询、可播放、可复用的日语学习内容。

| 字段 | 字段含义 | 数据类型语义 | 是否必填 | 默认值 | 枚举值 | 关系 | 唯一性约束 | 校验规则 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| resourceId | 资源唯一标识 | 文本型标识 | 是 | 系统生成或外部传入 | 无 | 被学习提交、音频生成、删除、详情查询引用 | 全局唯一 | 不能为空；更新、删除、详情查询时必须存在 |
| createdAt | 创建时间 | 日期时间 | 否 | 创建时系统生成 | 无 | 用于查询和排序 | 无 | 应为有效日期时间 |
| japaneseText | 日语展示文本 | 文本，当前长度上限为 1024 字符 | 否 | 空 | 无 | 翻译、日语朗读、游戏录入依赖该字段 | 无 | 执行翻译或日语朗读时应非空 |
| japaneseAudioFiles | 日语朗读音频列表 | 音频资源列表 | 否 | 空列表 | 无 | 包含多个音频资源 | 无 | 播放或课程合成时至少需要 1 个可播放音频 |
| chineseMeaning | 中文含义 | 文本，当前长度上限为 2048 字符 | 否 | 空 | 无 | 中文朗读和课程合成依赖该字段 | 无 | 执行中文朗读时应非空 |
| chineseAudioFiles | 中文朗读音频列表 | 音频资源列表 | 否 | 空列表 | 无 | 包含多个音频资源 | 无 | 课程合成时至少需要 1 个可播放音频 |
| source | 资源来源 | 短文本，当前长度上限为 255 字符 | 否 | 空 | 已确认值：“日语基础”“蓝宝书N1-文法”；是否固定待确认 | 用于查询、分组、卡片筛选 | 无 | 用于卡片视图时应能匹配基础来源 |

### 生命周期

1. 新建：用户手动新增，或学习游戏提交后自动生成。
2. 可用：资源可查询、查看、编辑、播放、翻译、生成音频或参与课程音频合成。
3. 更新：文本、释义、音频或来源可被修改。
4. 删除：用户确认删除后资源不再出现在查询结果中。

### 派生状态

| 状态 | 含义 | 进入条件 | 退出条件 |
| --- | --- | --- | --- |
| 文本待补全 | 缺少日语文本或中文含义 | 任一核心文本为空 | 补全缺失文本 |
| 无音频 | 日语和中文音频均为空 | 两个音频列表均为空 | 生成或上传任一音频 |
| 部分音频 | 只有日语音频或只有中文音频 | 仅一个音频列表非空 | 补齐另一种音频 |
| 可播放 | 有至少一个日语音频 | 日语音频列表非空 | 删除日语音频 |
| 可合成课程 | 有至少一个日语音频和一个中文音频 | 两个音频列表均非空 | 删除任一必要音频 |

## 实体二：音频资源

用于描述可播放、可引用的朗读音频。

| 字段 | 字段含义 | 数据类型语义 | 是否必填 | 默认值 | 枚举值 | 关系 | 唯一性约束 | 校验规则 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| audioId | 音频唯一标识 | 文本型标识 | 否 | 生成时产生 | 无 | 被日语学习资源引用 | 音频资源内唯一 | 存在时应唯一 |
| playableLocation | 可播放位置 | 可访问资源定位 | 是 | 无 | 无 | 用于在线播放 | 无 | 播放时必须可访问 |
| storageReference | 合成所需的资源引用 | 资源定位 | 否 | 无 | 无 | 用于课程音频合成 | 无 | 课程合成时必须存在 |
| fileType | 文件类型 | 音频格式标识 | 否 | 无 | 当前未确认固定枚举 | 无 | 无 | 应为可播放或可处理的音频格式 |
| createdAt | 创建时间 | 时间戳或日期时间 | 否 | 生成时产生 | 无 | 用于追溯 | 无 | 应为有效时间 |
| text | 音频对应文本 | 文本 | 否 | 空 | 无 | 与日语文本或中文含义对应 | 无 | 生成音频时建议记录 |
| voiceType | 发音类型或发音人 | 文本 | 否 | 未知 | 当前未确认固定枚举 | 无 | 无 | 可为空 |
| durationMillis | 音频时长 | 非负整数，单位毫秒 | 否 | 0 | 无 | 用于播放或统计参考 | 无 | 不得为负数 |

### 生命周期

1. 生成或上传：由用户上传、翻译结果携带或朗读生成能力产生。
2. 关联：加入日语学习资源的日语或中文音频列表。
3. 使用：被单条播放或课程音频合成引用。
4. 替换或删除：当前是否支持单独删除音频待确认。

## 实体三：学习统计

用于展示当前日语学习功能的学习累计结果。

| 字段 | 字段含义 | 数据类型语义 | 是否必填 | 默认值 | 枚举值 | 关系 | 唯一性约束 | 校验规则 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| statsId | 统计记录标识 | 文本型标识 | 是 | 系统初始化 | 无 | 被学习游戏读取和更新 | 当前为模块级唯一 | 必须存在 |
| createdAt | 统计开始时间 | 日期时间 | 是 | 初始化时生成 | 无 | 用于展示统计起点 | 无 | 应为有效日期时间 |
| totalLearnTimeSeconds | 累计学习时长 | 非负整数，单位秒 | 是 | 0 | 无 | 每次成功提交累加 | 无 | 不得为负 |
| totalSentenceCount | 累计学习句子数 | 非负整数 | 是 | 0 | 无 | 每次成功提交累加 | 无 | 不得为负 |
| todayLearnTimeSeconds | 今日学习时长 | 非负整数，单位秒 | 是 | 0 | 无 | 每日重置，每次成功提交累加 | 无 | 不得为负 |
| todaySentenceCount | 今日学习句子数 | 非负整数 | 是 | 0 | 无 | 每日重置，每次成功提交累加 | 无 | 不得为负 |
| todayDate | 今日统计日期 | 日期 | 是 | 当前日期 | 无 | 用于判断是否跨日 | 无 | 应能与当前日期比较 |

### 生命周期

1. 初始化：系统创建模块级统计记录，数量和时长均为 0。
2. 当日活跃：用户进入游戏或提交学习时，使用当前日期统计。
3. 跨日刷新：当记录日期不是当前日期时，今日统计归零并更新日期。
4. 提交累加：每次学习成功后累计和今日统计同步增加。

## 实体四：学习提交

学习提交是一次游戏录入动作的业务事件，当前不会作为独立长期资源保存，而是转化为日语学习资源并更新学习统计。

| 字段 | 字段含义 | 数据类型语义 | 是否必填 | 默认值 | 枚举值 | 关系 | 唯一性约束 | 校验规则 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| japaneseText | 用户输入的日语文本 | 文本 | 是 | 无 | 无 | 转化为日语学习资源 | 无 | 应非空；当前强制校验待确认 |
| japaneseAudioFiles | 日语朗读音频 | 音频资源列表 | 是 | 无 | 无 | 写入日语学习资源 | 无 | 至少 1 个可播放音频 |
| chineseMeaning | 中文翻译 | 文本 | 是 | 无 | 无 | 写入日语学习资源 | 无 | 应非空 |
| chineseAudioFiles | 中文朗读音频 | 音频资源列表 | 是 | 无 | 无 | 写入日语学习资源 | 无 | 至少 1 个可播放音频 |
| source | 本次学习来源 | 文本 | 是 | “蓝宝书N1-文法” | 是否可选待确认 | 写入日语学习资源 | 无 | 应有值 |
| elapsedSeconds | 本次学习耗时 | 非负整数，单位秒 | 是 | 0 | 无 | 累加到学习统计 | 无 | 不得为负 |

### 生命周期

1. 待输入。
2. 已输入，等待翻译和音频。
3. 已获得翻译和音频，播放日语原文。
4. 提交保存。
5. 成功后转化为日语学习资源并更新统计。
6. 失败后不应更新统计；是否保留失败记录待确认。
